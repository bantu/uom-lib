Welcome to Unit Systems, a selection of libraries and extension modules and unit types for JSR 385.
Using Indriya the Units of Measurement Reference Implementation.

The project currently contains the following modules:

  • Reusable Quantities
  • Common Unit Systems based on the RI
  • Unicode CLDR Unit System
  • Unified Code for Units of Measure (UCUM)
